Bienvenid@ a QuickHud

Welcome to QuickHud

QuickHud es un mod + plugin para Minecraft que permite a los administradores de servidor mostrar información personalizada en la pantalla de los jugadores, como contadores de jugadores vivos, kills, timers, coordenadas y equipos.

QuickHud is a mod + plugin for Minecraft that allows server administrators to display custom information on players' screens, such as alive player counters, kills, timers, coordinates and teams.

Mod + Plugin Fabric · Forge Bukkit · Spigot Minecraft 1.20.1

❓ ¿Qué es QuickHud?What is QuickHud?

QuickHud es un mod + plugin para Minecraft que permite a los administradores de servidor mostrar información personalizada en la pantalla HUD de los jugadores, como contadores de jugadores vivos, kills, temporizadores con cuenta regresiva, coordenadas y listas de equipo con skin.

QuickHud is a mod + plugin for Minecraft that allows server administrators to display custom information on the players' HUD screen, such as alive player counters, kills, countdown timers, coordinates, and team lists with skins.

🔌
Cómo funciona How it works El plugin (Bukkit/Spigot) corre en el servidor y controla qué se muestra. El mod (Fabric/Forge) debe estar instalado en el cliente de cada jugador para poder renderizar el HUD. The plugin (Bukkit/Spigot) runs on the server and controls what is shown. The mod (Fabric/Forge) must be installed on each player's client to render the HUD.

⌨️ Comandos del PluginPlugin Commands

Todos los comandos son ejecutados en el servidor por un operador (isOp). El estado se sincroniza automáticamente a todos los jugadores conectados.

All commands are executed on the server by an operator (isOp). State is automatically synced to all connected players.

❓ Preguntas FrecuentesFrequently Asked Questions

¿El HUD se muestra igual para todos los jugadores?
Is the HUD shown the same for all players?
Sí. QuickHud usa un estado global único. Cualquier cambio realizado por un admin se aplica instantáneamente a todos los jugadores conectados.
Yes. QuickHud uses a single global state. Any change made by an admin is instantly applied to all connected players.
¿El estado del HUD se guarda al reiniciar el servidor?
Is the HUD state saved on server restart?
Sí. El plugin guarda el estado en plugins/QuickHudPL/state.json automáticamente y lo restaura al iniciar.
Yes. The plugin automatically saves state to plugins/QuickHudPL/state.json and restores it on startup.
¿Los jugadores necesitan instalar algo?
Do players need to install anything?
Sí. Cada jugador debe tener el mod QuickHud instalado en su cliente (Fabric o Forge). Sin él, el HUD no se renderizará aunque el plugin esté activo en el servidor.
Yes. Each player must have the QuickHud mod installed on their client (Fabric or Forge). Without it, the HUD won't render even if the plugin is active on the server.
¿El temporizador se pausa al reiniciar?
Does the timer pause on restart?
Sí. Al recargar, el temporizador se restaura en estado pausado con los segundos restantes. Usa /hud timer resume para retomarlo.
Yes. On reload, the timer is restored in paused state with remaining seconds. Use /hud timer resume to continue it.
